Tema do Projeto : EcoBook (WIP)

O EcoBook é uma Aplicação Web desenhada para a partilha de livros. Um utilizador pode disponibilizar os seus livros para troca, indicando também uma lista com aqueles que gostaria de receber.

É também possível serem feitas doações de livros para o banco de livros EcoBook assim como requisições do mesmo.

Na altura de enviar os livros a EcoBook age como intermediário, recebendo os livros de ambos os clientes e reenviando à outra pessoa.

Para além disso, a EcoBook disponibiliza também um fórum para a troca de ideias sobre os livros, por parte de quem já leu ou de quem estaria interessado em ler.
